# Brand logo files

Drop the official logo images here (PNG with transparent or white background,
roughly 300px wide). The build picks them up automatically — no code change
needed — and shows a typographic placeholder for any file that is missing.

Expected filenames:

- `westin.png` — Westin Hotels & Resorts
- `hilton.png` — Hilton
- `le-meridien.png` — Le Méridien
- `aloft.png` — Aloft Hotels
- `sheraton.png` — Sheraton
- `kempinski.png` — Kempinski
- `the-paul.png` — The Paul
- `kumarakom.png` — Kumarakom Lake Resort
